# Tic Tac Toe


![Tic Tac Toe](tictactoe.png)

A basic Tic Tac Toe game built using pygame. No dependencies required.

## Install and Run

### Install 
```bash
git clone https://github.com/lucasmelosilva/tic_tac_toe
```

### Run
```bash 
cd ./tic_tac_toe

./tic_tac_toe
```
